This project involved developing a web application capable of real-time face mask detection to ensure public health safety compliance. Utilizing advanced AI and machine learning algorithms, the app analyzes video feeds to detect the presence of face masks on individuals. Its key features include high accuracy, low latency, and the ability to operate in various environments. This application serves as a crucial tool for businesses and public spaces to monitor and enforce mask-wearing protocols effectively.
